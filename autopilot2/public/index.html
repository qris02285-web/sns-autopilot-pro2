GitHub Pages で無料公開する手順（5分）
このHTMLファイル1つをアップロードするだけで、URLを知っている全員がアクセスできるWebアプリになります。サーバー代・月額費用は一切かかりません。
01
GitHubアカウントを作成
github.com にアクセスして「Sign up」→ メールアドレスで無料登録（Googleアカウントでも可）
02
新しいリポジトリを作成
ログイン後、右上「+」→「New repository」→ Repository nameにsns-autopilotと入力 → 「Public」を選択 → 「Create repository」
03
HTMLファイルをアップロード
このファイル（sns_autopilot_final.html）をindex.htmlにリネーム → リポジトリページの「uploading an existing file」をクリック → ドラッグ&ドロップ → 「Commit changes」
04
GitHub Pages を有効化
リポジトリの「Settings」タブ → 左メニュー「Pages」→ Branch: main → folder: /(root) → 「Save」
05
URLをシェア
1〜2分後にhttps://あなたのユーザーID.github.io/sns-autopilot/でアクセス可能になります。このURLを知人・SNSでシェアするだけで誰でも使えます。
💡 ポイント： 全データはブラウザの localStorage に保存されます。他の人が使っても、あなたの設定・投稿履歴には影響しません。それぞれが独立したデータを持ちます。
